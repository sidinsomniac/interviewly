# Interviewly

**An AI-driven Teams interviewer bot that fills out the Publicis Sapient Experience Engineering hiring probe form.**

This is the planning home for **App 1** in the PS Expo: AI@Work 2026 submission. Interviewly conducts technical interviews via Microsoft Teams, generates probe-form-aligned questions from a JD, posts them in the meeting chat, captures the conversation via Teams' built-in transcription, and emits a filled `.xlsx` probe form for downstream evaluation by **Verdict** (App 2 — separate project).
